
## API

属性如下

| 属性          | 说明             | 类型             | 默认值  |
| ------------- | ---------------- | ---------------- | ------- |
| value         | 当前色值     | string | `#1890FF`       |
| width         | 宽度 | string        | `auto`       |
| edit | 是否可编辑     | edit          | `true` |
| placement | 子选择框位置     | `top` 、 `bottom`          | `top` |
| disabled | 是否禁用     | boolean          | `false` |
| onChange      | 改变回调     | function          | `(color)=>{//返回颜色值}` |


> 注意：其他属性请参阅antd
