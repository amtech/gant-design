
## API

自动刷新组件

| 属性      | 说明                 | 类型     | 默认值     |
| --------- | -------------------- | -------- | ---------- |
| auto      | 是否开启自动刷新     | boolean  | `true`     |
| interval  | 刷新间隔，单位（分） | number   | `1`        |
| refresh   | 刷新触发的回调方法   | function | ()=>{}     |
| time      | 自定义显示的时间     | string   | `当前时间` |
| className | 类名                 | string   |            |
| style     | 样式                 | object   |            |
