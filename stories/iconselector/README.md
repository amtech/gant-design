
## API

属性如下

| 属性     | 说明                                      | 类型                         | 默认值 |
| -------- | ----------------------------------------- | ---------------------------- | ------ |
| onBlur   | 失焦回调                                  | Function                     | 'line' |
| value    | 值，AntIcon的type 或者iconfont的className | string                       |        |
| onChange | 选择回调                                  | Function(icon: string): void |        |

## 注意
如需选择iconfont 需要前置引入iconfont图标
