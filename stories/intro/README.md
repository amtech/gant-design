
## API

简介

| 属性        | 说明     | 类型             | 默认值     |
| ----------- | -------- | ---------------- | ---------- |
| title       | 标题     | string\ReactNode | `当前时间` |
| content     | 内容     | string\ReactNode |            |
| image       | 图片url  | string           |            |
| imageAlign  | 图片位置 | 'left','right'   | 'left'     |
| imageRadius | 图片圆角 | number           | 0          |
